匿名公共聊天 — 无需注册，输入昵称即可聊天。

## 功能特点

| 功能 | 描述 |
|------|------|
| **无需注册** | 输入昵称即可聊天 |
| **频道** | 创建任意频道，24小时无活动自动删除 |
| **历史** | 每个频道保留最近50条消息 |
| **联邦** | 服务器之间同步消息 |
| **匿名** | 无账号，无追踪 |
| **移动端** | 完全响应式设计 |

## 快速开始

```bash
git clone https://github.com/HepBHbIu/typechat.git
cd typechat
npm install
npm start
```

打开 http://localhost:3001

## 如何使用

1. 打开页面
2. 输入任意昵称
3. 选择或创建频道
4. 开始聊天！
